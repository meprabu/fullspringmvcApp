Crisis Stabilization Unit Home

This is a data collection resource for Bed count. This allows DMH and its certified providers to administer and report the number of available beds. Menu availability is based on permissions for each user.

    BC Licensed/Active Beds
    Occupied Beds
    Reports

BC Licensed/Active Beds - Administrators maintain the number of licensed beds and the number of active beds.

BC Occupied Beds - Providers submit the number of occupied beds.

BC Report - Report provides the summary data based on entries by the providers.

DMH Website - This link leads to the DMH website, www.dmh.ms.gov. This will navigate away from the Bed Count resource.

CDR Main Menu - This link will return the user to the Central Data Repository (CDR).
